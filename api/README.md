# Node Server Whit Babel

## Getting Started

First lest install all the dependecies running

```
$ yarn
```

Then run the server

```
$ yarn start
```

Run the test

```
$ yarn test
```

Build the Project

```
$ yarn build
```

### Prerequisites

- [Node.js](https://nodejs.org/es/)
- [Yarn](https://yarnpkg.com)
